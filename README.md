# Critical-Point
BUL: Това е официалната страница на проекта "Критична Точка", направен за олимпиада по информационни технологии - 2025 година. В нея играчът използва портали, за да се придвижва в пространството и да решава логически пъзели. Използвани софтуери - Unreal Engine, Blender, InstaMAT и други.

ENG: This is a game made in Unreal Engine for a competition. In it the player uses portals to move through space and solve logic puzzles. Used software - Unreal Engine, Blender, InstaMAT and others.
