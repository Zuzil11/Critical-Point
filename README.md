# critical-point
BUL: Това е официалната страница на проекта "Критична Точка", направен за олимпиада по информационни технологии - 2025 година. Играта е все още в процес на разработка и ще е готова след около няколко месеца. В нея главно се използва основната механиката от игрите Portal и Portal 2, където играчът използва портали, за да се придвижва в пространството и да решава логически пъзели. Използвани софтуери - Unreal Engine, Blender, BandLab и други.

ENG: This is a game made in Unreal Engine for a competition. It mainly uses the core mechanics from the games Portal and Portal 2, where the player uses portals to move through space and solve logic puzzles. Please leave a suggestion!
